Parameter :id

Semua route menggunakan parameter :id untuk mengidentifikasi data yang akan ditampilkan.

Contoh: /report/invoice/123 akan menampilkan invoice dengan ID 123.

Printer Compatibility

Semua report sudah dioptimasi untuk printer dot matrix Epson LX-310 dengan:

  • Font: Courier New (monospace)
  • Border: 1px (tipis)
  • Font size: 9-11px

Blangko

Surat Jalan PUS1 dan PUS2 menggunakan blangko pre-printed dengan ukuran khusus:

  • PUS1: 21.5cm x 14cm (blangko kuning)
  • PUS2: 21.5cm x 16cm (blangko merah muda)
